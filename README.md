 # cookie clicker game 

 A simple clicker game built with HTML, CSS, and Javasript. Click to earn cookies, buy upgrades to increase your cookies per second.

 # How to play

 - Click the cookie to earn cookies.
 - Earn cookies automatically based on your CPS ( count per second).
 - Buy upgrades from the shop to increase your CPS.
 - Your progress is saved automatically using localStorage.
 
 # Features

 - Clickable cookies with animation.
 - Shop system with upgrades.
 - CPS ( cookies per second).
 - Persistent saved data using localStorage.
 - Responsive layout with shop on the right.
